## 🎯 Key Features

### ✅ 6 Hand Gesture Modes
| Mode | Gesture | Audio Effect |
|------|---------|---|
| 0 | Open Hand | Complex modulation |
| 1 | Closed Fist | Basic sine/square |
| 2 | Fist + Thumb | Sub-octave |
| 3 | Fist + Index | Overtones |
| 4 | Both Extended | Combined effects |
| 5 | Peace Sign | Reverb/Delay |

### ✅ Continuous Modulation
- **Hand Roll** (left/right tilt) → Vibrato control
- **Hand Pitch** (forward/back) → Expression control
- **Distance** (close/far) → Pitch frequency

### ✅ Smart Smoothing
- Moving average filters (5-sample window)
- Jitter reduction without latency
- Configurable smoothing levels

### ✅ Safety Features
- Auto-mute when hand leaves frame
- Emergency panic mode
- Graceful shutdown sequence
- Input validation & clamping

## 📡 OSC Control Surface

### 6 Real-Time Parameters
```
/synth/pitch       0.0 to 1.0    (distance sensor)
/synth/mode        0 to 5        (gesture state)
/synth/vibrato     0.0 to 1.0    (hand roll)
/synth/expression  0.0 to 1.0    (hand pitch)
/synth/volume      0.0 to 1.0    (safety gate)
/synth/panic       0 or 1        (emergency stop)
```

### Update Rate
- **Continuous:** 30 FPS for pitch/vibrato/expression
- **Event-driven:** Mode changes only on state change
- **Safety:** Volume updates only on hand detection change
- **Network:** <1ms latency (localhost UDP)
